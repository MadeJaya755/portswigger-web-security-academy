# Authentication Testing

This folder contains scripts and proof-of-work from solving
PortSwigger Web Security Academy labs related to authentication flaws.

The focus is on exploiting weaknesses in authentication mechanisms
rather than describing theoretical concepts.

---

## Approach

- Manual request analysis using Burp Suite
- Identification of logic flaws in login and account workflows
- Brute-force and bypass techniques where applicable
- Automation of repetitive attacks using Python

---

## Tools

- Burp Suite
- Python

---

## Notes

Documentation is intentionally minimal.
Most labs are represented by scripts and screenshots as proof-of-work.
