# API Security Testing

This folder contains scripts and proof-of-work from solving
PortSwigger Web Security Academy API-related labs.

The focus is on practical exploitation of real-world API
authorization and logic flaws.

---

## Approach

- Manual testing using Burp Suite
- Identification of authorization and logic issues
- Automation of repetitive attacks using Python

---

## Tools

- Burp Suite
- Python

---

## Notes

Documentation is intentionally kept minimal.
